# 真实可整合 PT 资源站功能地图

## 目标定位

本项目面向“真实可整合 PT 资源站”场景，核心目标是把用户已授权访问的 PT、Torznab、Jackett、Prowlarr、RSS、API、私有库等来源统一接入，完成资源发现、标准化解析、质量排序、订阅通知与运维治理。

系统不提供、不过度描述、也不鼓励任何绕过授权、破解访问限制、规避站点规则或侵权抓取的实现方式。所有来源必须满足以下前提：

- 用户或部署方对来源拥有合法访问权限。
- 接入方式遵守来源站点、API、RSS、Torznab、Jackett、Prowlarr 或私有库的使用规则。
- 不存储或分发无授权内容本体，仅处理资源元数据、索引信息、订阅状态和用户配置。
- 对敏感凭据、站点令牌、Cookie、API Key 等信息进行加密存储和最小化使用。

## 用户端

### 账号与个人设置

- 用户注册、登录、退出、会话续期。
- 个人资料维护：昵称、头像、语言、时区、默认资源偏好。
- 多因素认证可选支持。
- API Token 或 Webhook Token 的个人级管理。
- 下载器、通知渠道、订阅偏好的个人配置。
- 账号安全记录：登录历史、活跃设备、异常登录提醒。

### 资源搜索

- 统一搜索入口，支持按关键词、分类、年份、标签、站点、质量、体积、发布时间筛选。
- 支持精确搜索、模糊搜索、别名搜索、标题归一化搜索。
- 搜索结果聚合展示，按资源、来源、质量、发布时间、热度等维度组织。
- 支持高级筛选：分辨率、编码、音频格式、字幕、媒介来源、发布组、促销状态、可用性状态。
- 支持搜索历史与常用筛选条件保存。

### 资源详情

- 展示资源标题、原始标题、别名、分类、来源站点、发布时间、体积、质量标签、文件列表摘要、描述信息。
- 展示种子健康度或可用性摘要，但不暴露来源不允许公开的信息。
- 展示站点规则提示、下载限制、促销状态、做种要求等合规提醒。
- 展示同一资源的多来源、多版本对比。
- 支持收藏、标记已看、标记不感兴趣、加入订阅规则。

### 订阅与追踪

- 影视、剧集、动漫、音乐、软件、电子书等分类订阅。
- 支持按标题、别名、季集、年份、分辨率、质量、发布组、字幕等条件订阅。
- 支持订阅命中记录、自动匹配记录和忽略记录。
- 支持订阅优先级、黑白名单、延迟通知、重复资源去重。
- 支持订阅暂停、恢复、归档、手动重跑。

### 下载与外部工具联动

- 与用户授权的下载器或媒体管理工具集成，例如通过受控 API 下发任务。
- 支持下载前确认、自动下载、仅通知不下载三种策略。
- 支持下载任务状态回传：已添加、下载中、完成、失败、已暂停。
- 支持按用户、来源、分类设置下载目录映射。
- 支持任务失败重试、重复任务拦截、下载器不可用提醒。

### 通知中心

- 站内通知列表。
- 邮件、Webhook、Telegram、企业微信、Discord、Bark、ServerChan 等通知渠道扩展。
- 订阅命中通知、同步失败通知、凭据失效通知、规则冲突通知。
- 通知模板配置与通知频率限制。
- 通知已读、归档、重发、静默时段。

### 用户体验

- 首页资源流：最新、热门、推荐、订阅命中、最近更新。
- 响应式布局，支持桌面端与移动端浏览。
- 暗色模式与紧凑视图。
- 资源卡片、列表、表格多种展示模式。
- 空状态、加载状态、错误状态清晰可见。
- 对站点权限不足、来源失效、订阅未命中等情况提供可理解的提示。

## 管理端

- RBAC 权限模型：超级管理员、站点管理员、审计员、普通用户等。
- 权限粒度覆盖来源配置、用户管理、任务管理、系统设置、审计查看。

### 用户管理

- 用户配额管理：搜索频率、订阅数量、来源数量、通知频率。
- 用户访问来源的授权范围管理。
- 用户异常行为查看：高频请求、失败登录、异常下载触发。
- 用户数据导出与删除请求处理。

### 来源管理

- 新增、编辑、禁用、删除授权来源。
- 来源类型包括 PT、Torznab、Jackett、Prowlarr、RSS、API、私有库。
- 来源健康检查：连通性、认证状态、响应时间、错误率。
- 来源字段映射配置：分类、标签、质量、发布时间、体积、详情链接、唯一标识。
- 来源级限流、同步频率、重试策略、失败熔断。
- 来源访问凭据的加密存储与轮换。

### 分类与元数据管理

- 全局分类体系：电影、剧集、动漫、音乐、游戏、软件、电子书、纪录片、体育等。
- 来源分类到全局分类的映射。
- 质量标签标准化：分辨率、编码、音频、字幕、媒介来源、HDR、杜比视界等。
- 标题别名库维护。
- 发布组、地区、语言、字幕标签维护。

### 规则管理

- 订阅匹配规则管理。
- 质量排序规则管理。
- 黑白名单管理。
- 去重规则管理。
- 通知规则管理。
- 来源优先级与回退策略管理。

### 审计与合规

- 管理端操作日志。
- 用户关键操作日志。
- 来源访问日志摘要。
- 凭据使用记录。
- 数据删除、导出、脱敏记录。
- 合规公告与站点接入规则配置。

## 来源接入

### 支持来源类型

- 有权限访问的 PT 站点。
- Torznab 兼容接口。
- Jackett 聚合接口。
- Prowlarr 聚合接口。
- RSS 订阅源。
- 官方或授权 API。
- 私有资源库或内部索引库。

### 接入原则

- 所有接入必须显式配置授权凭据或授权访问地址。
- 不内置绕过登录、绕过限流、绕过 Cloudflare、破解验证码或模拟非法访问的逻辑。
- 不内置未授权站点抓取模板。
- 默认尊重来源限流、缓存策略和使用条款。
- 支持来源级别的启用状态、同步频率、请求超时和错误阈值。

### 通用接入能力

- 统一来源适配器接口。
- 连接测试与认证测试。
- 来源能力探测：是否支持搜索、分类、分页、详情、增量同步。
- 来源字段归一化。
- 来源返回数据校验。
- 来源错误分类：认证失败、限流、超时、格式错误、服务不可用、权限不足。
- 来源同步水位记录。

### Torznab / Jackett / Prowlarr

- 配置 Base URL、API Key、分类映射和超时参数。
- 支持按能力调用搜索、分类查询、最新资源查询。
- 支持多实例配置。
- 支持聚合器返回的来源标识保留。
- 支持按来源、分类、标签设置优先级。

### RSS

- 配置 RSS URL、认证参数、分类映射和刷新周期。
- 支持 RSS 条目去重。
- 支持 RSS 字段标准化。
- 支持发布时间、标题、链接、描述、体积等字段解析。
- 支持 RSS 源失效检测。

### API / 私有库

- 支持标准 REST API 或内部服务 API。
- 支持 Token、Basic Auth、签名鉴权等授权方式。
- 支持自定义字段映射。
- 支持增量同步、全量同步和按需搜索。
- 支持私有库元数据可信等级配置。

## 同步任务

### 任务类型

- 来源健康检查任务。
- 最新资源增量同步任务。
- 全量索引同步任务。
- 订阅规则匹配任务。
- 元数据补全任务。
- 质量评分重算任务。
- 通知投递任务。
- 数据清理与归档任务。

### 调度能力

- Cron 表达式调度。
- 固定间隔调度。
- 手动触发任务。
- 来源变更后触发任务。
- 订阅规则变更后触发重算。
- 失败任务延迟重试。
- 任务并发数控制。

### 任务状态

- 待执行、执行中、成功、失败、取消、超时、跳过。
- 记录任务开始时间、结束时间、耗时、处理数量、成功数量、失败数量。
- 记录任务错误摘要与可追踪上下文。
- 支持任务日志查看与下载。
- 支持任务重跑、取消和暂停。

### 数据一致性

- 来源同步水位控制。
- 幂等写入，避免重复资源。
- 批量写入与事务保护。
- 任务锁，避免同一来源重复同步。
- 失败批次可重试。
- 资源删除或失效状态同步。

- 原始标题。
- 标准标题。
- 别名。
- 分类。
- 年份。
- 季集信息。
- 发布时间。
- 体积。
- 文件数量与文件列表摘要。
- 分辨率。
- 视频编码。
- 音频编码。
- 字幕语言。

- 支持人工修正标准标题、别名和分类。
- 支持补全失败记录。
- 支持元数据来源可信等级。

- 低置信度命中进入待确认队列。

## 安全合规

### 授权与访问控制

- 用户认证与会话管理。
- RBAC 权限控制。
- 来源访问权限隔离。
- 管理端敏感操作二次确认。
- API Token 权限范围与有效期控制。

### 凭据安全

- API Key、Cookie、Token 等敏感信息加密存储。
- 敏感信息在日志、错误信息、前端页面中脱敏。
- 支持凭据轮换。
- 支持凭据失效检测与提醒。
- 最小权限原则。

### 数据合规

- 只保存资源元数据、订阅配置、任务日志和必要的审计记录。
- 不存储、缓存或分发无授权内容本体。
- 支持用户数据导出。
- 支持用户数据删除。
- 支持日志保留周期配置。
- 支持隐私政策、使用条款和来源合规说明。

### 来源合规

- 仅允许配置有权限的 PT、Torznab、Jackett、Prowlarr、RSS、API、私有库。
- 尊重来源站点规则、限流策略和授权范围。
- 禁止绕过访问控制、验证码、登录限制或反滥用机制。
- 管理端展示来源接入责任提示。

### 风险控制

- 登录失败限制。
- API 请求限流。
- 来源请求限流。
- 异常访问检测。
- 管理端危险操作审计。
- 安全事件告警。
